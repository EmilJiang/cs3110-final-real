# Installation Steps:

1. Navigate to the final_project folder in terminal

2. Type in "opam install pyml"

3. "python3 -m venv venv"

4. "source venv/bin/activate"

5. "pip3 install openai"

6. Type in "dune exec bin/main.exe" into the terminal

Note: This terminal integration does not allow multiline inputs. This restraint 
is only valid for terminal. Any other UI usage multiline is enabled.
